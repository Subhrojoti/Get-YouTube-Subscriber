API Endpoint Documentation: Retrieving Subscriber Data

Use the following endpoint to retrieve an array of all subscribers:

Retrieve All Subscribers:

• GET http://localhost:3000/subscribers

Response with An array of Subscriber Objects

Retrieve Subscribers' Names and Channels

• GET http://localhost:3000/subscribers/names

Response with An array of subscriber objects containing only the "name" and "subscribedChannel" fields.

Retrieve a Subscriber by ID:

• GET http://localhost:3000/subscribers/:id

Parameters

:id:The unique identifier of the subscriber.

Responses

• A single subscriber object corresponding to the provided ID.

• If the provided ID does not match any subscriber, the server responds with
a status code 400 & an error message in the format { "message": "error.message" }.
